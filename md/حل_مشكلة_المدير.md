# ✅ حل: No manager email found

## المشكلة
عند رفع طلب إجازة، تظهر رسالة:
```
⚠️ No manager email found - notification cannot be sent
```

هذا يعني أن الموظف **ليس له مدير محدد** في النظام.

---

## الحل: تحديد المدير المباشر من admin.html

### الخطوة 1: افتح صفحة لوحة التحكم

1. افتح `admin.html` في المتصفح
2. سجّل دخول بحساب المدير
3. اضغط على **📧 إدارة الإيميلات** من القائمة الجانبية

### الخطوة 2: حدّد المدير لكل موظف

ستظهر قائمة بجميع الموظفين مع أعمدة:
- الاسم
- الرقم الوظيفي
- الفرع
- البريد الإلكتروني
- **المدير المباشر** ← العمود الجديد
- الإجراءات

#### لتحديد مدير لموظف:

1. ابحث عن الموظف في القائمة
2. اضغط زر **➕ مدير** (أو **✏️ مدير** للتعديل)
3. ستظهر نافذة بقائمة جميع الموظفين المتاحين
4. أدخل **الرقم الوظيفي** للمدير المباشر
5. اضغط OK

**مثال:**
```
الموظف: أحمد علي
الرقم الوظيفي: E001

أدخل الرقم الوظيفي للمدير المباشر: M001
```

---

## مثال عملي

### السيناريو:
- **المدير:** محمد خالد (M001) - `mohammed@company.com`
- **الموظفون تحت إدارته:**
  - أحمد (E001) - `ahmad@company.com`
  - فاطمة (E002) - `fatima@company.com`
  - سالم (E003) - `salem@company.com`

### الخطوات:

#### 1. حدد المدير لأحمد (E001)
```
اضغط: ➕ مدير بجانب أحمد
أدخل: M001
✅ تم تحديد المدير المباشر بنجاح
```

#### 2. حدد المدير لفاطمة (E002)
```
اضغط: ➕ مدير بجانب فاطمة
أدخل: M001
✅ تم تحديد المدير المباشر بنجاح
```

#### 3. حدد المدير لسالم (E003)
```
اضغط: ➕ مدير بجانب سالم
أدخل: M001
✅ تم تحديد المدير المباشر بنجاح
```

الآن جميع الموظفين الثلاثة سيُرسل طلباتهم إلى بريد محمد (M001).

---

## التحقق من النجاح

### بعد تحديد المدراء:

1. سجّل دخول كموظف في `hr-management.html`
2. قدّم طلب إجازة جديد
3. تحقق من Console (F12) - يجب أن ترى:
   ```
   ✅ Leave request notification sent to manager
   ```
4. تحقق من بريد المدير - يجب أن يصله إشعار

## نصائح مهمة

### ✅ التسلسل الإداري الصحيح

```
المدير العام (CEO)
  ↓
مدير الفرع 1        مدير الفرع 2
  ↓                      ↓
موظفين الفرع 1      موظفين الفرع 2
```

**مثال SQL:**
```sql
-- تعيين مدير الفرع
UPDATE employees SET manager_id = 'CEO001' WHERE employee_id = 'M001';
UPDATE employees SET manager_id = 'CEO001' WHERE employee_id = 'M002';

-- تعيين الموظفين تحت مدراء الفروع
UPDATE employees SET manager_id = 'M001' WHERE employee_id IN ('E001', 'E002', 'E003');
UPDATE employees SET manager_id = 'M002' WHERE employee_id IN ('E004', 'E005', 'E006');
```

### ⚠️ أخطاء شائعة

#### 1. الموظف يحدد نفسه كمدير
```
❌ لا يمكن للموظف أن يكون مديراً لنفسه!
```
**الحل:** أدخل رقم وظيفي لشخص آخر.

#### 2. الرقم الوظيفي غير موجود
```
❌ الرقم الوظيفي "M999" غير موجود في قاعدة البيانات
```
**الحل:** تأكد من الرقم الوظيفي صحيح وموجود في النظام.

#### 3. المدير ليس له بريد إلكتروني
```
⚠️ No manager email found
```
**الحل:** أضف بريد إلكتروني للمدير من نفس الصفحة (زر **➕ إيميل**).

## التحقق من البيانات في SQL

### عرض جميع الموظفين ومدرائهم:

```sql
SELECT 
    e.employee_id,
    e.name AS employee_name,
    e.email AS employee_email,
    e.manager_id,
    m.name AS manager_name,
    m.email AS manager_email
FROM employees e
LEFT JOIN employees m ON e.manager_id = m.employee_id
ORDER BY e.name;
```

### عرض الموظفين بدون مدير:

```sql
SELECT employee_id, name, email, branch_name
FROM employees
WHERE manager_id IS NULL OR manager_id = ''
ORDER BY name;
```

### عرض المدراء وعدد الموظفين تحتهم:

```sql
SELECT 
    m.employee_id AS manager_id,
    m.name AS manager_name,
    m.email AS manager_email,
    COUNT(e.employee_id) AS employees_count
FROM employees m
LEFT JOIN employees e ON e.manager_id = m.employee_id
WHERE m.employee_id IN (SELECT DISTINCT manager_id FROM employees WHERE manager_id IS NOT NULL)
GROUP BY m.employee_id, m.name, m.email
ORDER BY employees_count DESC;
```

## الخلاصة

### ما يحتاجه كل موظف:
1. ✅ بريد إلكتروني (email)
2. ✅ مدير مباشر (manager_id)

### ما يحتاجه كل مدير:
1. ✅ بريد إلكتروني (email)
2. ✅ قد يحتاج مدير أعلى منه (manager_id) حسب الهيكل التنظيمي

### عند رفع طلب إجازة:
- يُرسل الإشعار لـ **بريد المدير المباشر**
- إذا لم يكن للموظف manager_id → لن يُرسل الإشعار
- إذا لم يكن للمدير email → لن يُرسل الإشعار
