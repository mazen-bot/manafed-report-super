# 📋 دليل نظام الملفات الطبية للإجازات المرضية
## Medical Certificate Management System Guide

---

## 📌 نظرة عامة

تم إضافة نظام شامل لإدارة الملفات الطبية عند طلب إجازة مرضية، يتضمن:
- ✅ إلزامية رفع شهادة طبية
- ✅ مدة محددة للرفع (3 أيام)
- ✅ نظام تنبيهات لتجنب الخصم
- ✅ تطبيق خصم تلقائي عند عدم الرفع

---

## 🎯 المزايا الرئيسية

### للموظف:
1. **واجهة سهلة الاستخدام** لرفع الملفات الطبية
2. **تنبيهات واضحة** عند اقتراب انتهاء المدة
3. **تنوع الملفات** المدعومة (صور + PDF)
4. **متابعة سهلة** لحالة الملفات المرفوعة

### للإدارة:
1. **متابعة الامتثال** لسياسة الملفات الطبية
2. **سجل شامل** للملفات المرفوعة والخصومات
3. **سهولة التحقق** من الملفات المفقودة

---

## 📋 خطوات الاستخدام

### 1️⃣ تقديم طلب إجازة مرضية

```
1. اذهب إلى تبويب "➕ طلب جديد"
2. اختر "إجازة مرضية" من نوع الطلب
3. أدخل التواريخ والسبب
4. اضغط "📤 إرسال الطلب"
```

**ملاحظة**: عند إرسال طلب إجازة مرضية، يتم حساب الموعد النهائي (3 أيام من تاريخ الطلب) تلقائياً.

### 2️⃣ رفع الشهادة الطبية

```
1. انقر على تبويب "📄 الملفات الطبية"
2. ستظهر جميع طلبات الإجازة المرضية المعلقة
3. اختر طلب وانقر "📤 رفع الشهادة الطبية"
4. اختر الملف من جهازك (صورة أو PDF)
5. انتظر تأكيد الرفع ✅
```

### 3️⃣ المتابعة والتنبيهات

يتم عرض حالة كل طلب إجازة مرضية مع:
- **عدد الأيام المتبقية** للرفع
- **تنبيهات تحذيرية** عند اقتراب الانتهاء
- **حالة الملف** (مرفوع/لم يتم الرفع)

## 💾 الملفات المدعومة

### الصيغ المقبولة:
- 📷 صور: JPEG, PNG, GIF
- 📄 وثائق: PDF

### متطلبات الملف:
- **الحد الأقصى**: 5 MB
- **الجودة**: واضحة وقابلة للقراءة
- **المحتوى**: شهادة طبية أصلية

---

## ⚠️ نظام الخصم

### شروط تطبيق الخصم:

1. **طلب إجازة مرضية** مُرسل من الموظف
2. **عدم رفع الملف** خلال 3 أيام من تاريخ الطلب
3. **انتهاء الموعد النهائي** تلقائياً

### مبلغ الخصم:
- **50 درهم / يوم** من الإجازة المرضية
- **مثال**: 3 أيام بدون ملف = 150 درهم خصم

### السجل:
- يتم حفظ جميع الخصومات في جدول `penalty_logs`
- يمكن للإدارة الاطلاع على سجل الخصومات
- يتم توثيق السبب والتاريخ تلقائياً

## 🔒 الأمان والصلاحيات

### Row Level Security (RLS):

✅ **الموظف**:
- ✓ يرى ملفاته فقط
- ✓ يرفع ملفاته فقط
- ✓ يحذف ملفاته فقط

✅ **المدير**:
- ✓ يرى جميع الملفات
- ✓ يُعدِّل السجلات
- ✓ يراجع الخصومات

### تخزين الملفات:
- تُرفع الملفات إلى **Supabase Storage** (بدلاً من قاعدة البيانات)
- روابط آمنة للتحميل والعرض
- حذف تلقائي عند حذف الطلب

---

## 📞 العمليات المشروطة

### عند رفع إجازة مرضية:
```
1. إنشاء سجل طلب
2. تحديد الموعد النهائي = اليوم + 3 أيام
3. تعيين حالة attachment_uploaded = false
4. توجيه الموظف لرفع الملف
```

### عند رفع ملف:
```
1. التحقق من نوع الملف
2. التحقق من حجم الملف (< 5MB)
3. الرفع إلى Supabase Storage
4. حفظ رابط الملف في قاعدة البيانات
5. تحديث حالة attachment_uploaded = true
6. تسجيل تاريخ الرفع
```

### عند انتهاء المدة (3 أيام):
```
1. البحث عن إجازات مرضية معلقة بدون ملفات
2. التحقق من تجاوز الموعد النهائي
3. تطبيق الخصم تلقائياً
4. تحديث حالة penalty_applied = true
5. تسجيل الخصم في جدول penalty_logs
```

## ❓ الأسئلة الشائعة

### س: ماذا لو رفعت ملفاً خاطئاً؟
**ج**: يمكنك حذف الملف وإعادة رفع ملف جديد قبل انتهاء المدة.

### س: هل يمكنني رفع الملف بعد انتهاء المدة؟
**ج**: نعم، يمكنك الرفع بعد انتهاء المدة، لكن سيتم تطبيق الخصم.

### س: كم الخصم تحديداً؟
**ج**: 50 درهم لكل يوم إجازة مرضية. مثال: 3 أيام = 150 درهم.

### س: كيف أعرف أن الملف تم رفعه بنجاح؟
**ج**: ستظهر رسالة تأكيد "✅ تم رفع الشهادة الطبية بنجاح" وسيتحول الزر للأزرق مع "✅ تم الرفع".

### س: هل أحتاج لرفع ملف لكل يوم من الإجازة؟
**ج**: لا، ملف واحد يكفي لطلب الإجازة بأكمله.

### س: ماذا لو فقدت الملف الأصلي؟
**ج**: تواصل مع الإدارة، قد تتمكن من السماح برفع لاحق.

## 📝 ملاحظات مهمة

⚠️ **الموعد النهائي 3 أيام**:
- يُحسب من تاريخ تقديم الطلب
- يشمل أيام نهاية الأسبوع والعطلات
- لا يتأثر بساعات العمل

✅ **نوع الملف**:
- يُفضل رفع صورة واضحة من الشهادة الطبية
- يمكن رفع ملف PDF مباشرة من المستشفى

⚠️ **التخزين**:
- الملفات تُحفظ في Supabase Storage بشكل آمن
- يمكن تحميل الملفات لاحقاً
- الملفات المحذوفة لا يمكن استعادتها
