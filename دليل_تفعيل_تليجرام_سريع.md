## 🚀 الخطوات (15 دقيقة فقط!)

### 1️⃣ إنشاء Bot في تليجرام (3 دقائق)

1. افتح تطبيق تليجرام
2. ابحث عن: `@BotFather`
3. افتح المحادثة واضغط **Start**
4. أرسل: `/newbot`
5. اكتب اسم البوت مثل: `Manafed HR System`
6. اكتب username للبوت (يجب ينتهي بـ bot): `manafed_hr_bot`
7. **سيعطيك Bot Token** - احفظه! مثل:
   ```
   123456789:ABCdefGHIjklMNOpqrsTUVwxyz
   ```

### 2️⃣ إضافة عمود telegram_chat_id (2 دقيقة)

افتح Supabase SQL Editor وشغّل:

```sql
-- إضافة عمود Chat ID
ALTER TABLE employees 
ADD COLUMN IF NOT EXISTS telegram_chat_id TEXT;

-- إضافة index لسرعة البحث
CREATE INDEX IF NOT EXISTS idx_employees_telegram_chat_id 
ON employees(telegram_chat_id);

-- تحديث العمود
COMMENT ON COLUMN employees.telegram_chat_id 
IS 'Telegram botToken: '123456789:ABCdefGHIjklMNOpqrsTUVwxyz' // ضع التوكن الخاص بكbotToken: '123456789:ABCdefGHIjklMNOpqrsTUVwxyz' // ضع التوكن الخاص بك للإشعارات';
```

### 3️⃣ جمع Chat IDs للموظفين (5 دقائق)

**الطريقة السهلة:**

1. كل موظف يفتح البوت الخاص بك: `@manafed_hr_bot`
2. يضغط **Start**
3. يفتح في تليجرام: `@userinfobot`
4. يضغط **Start**
5. سيعطيه Chat ID مثل: `987654321`
6. يعطيك الرقم أنت

**إدخال Chat IDs في النظام:**

افتح `admin.html` → انتقل لصفحة **إدارة الإيميلات والمديرين** → سنضيف عمود Chat ID لاحقاً

أو **مؤقتاً** استخدم SQL:

```sql
UPDATE employees 
SET telegram_chat_id = '987654321' 
WHERE employee_id = '1001';
```

### 4️⃣ تفعيل الكود (3 دقائق)

#### أ) افتح `telegram-notifications.js`

غيّر السطر 8:

```javascript
botToken: 'YOUR_BOT_TOKEN_HERE', // ❌ قبل
```

إلى:

```javascript
botToken: '123456789:ABCdefGHIjklMNOpqrsTUVwxyz', // ✅ بعد (ضع Token الخاص بك)
```

#### ب) افتح `hr-management.html`

أضف بعد السطر الذي يحمّل `emailjs-notifications.js`:

```html
<!-- إشعارات تليجرام -->
<script src="telegram-notifications.js"></script>
```

#### ج) في دالة `submitLeaveRequest()` أضف:

```javascript
// إرسال إشعار تليجرام للمدير
if (window.sendLeaveRequestTelegram) {
    await sendLeaveRequestTelegram(leaveData);
}
```

#### د) في دالة `updateRequestStatus()` أضف:

```javascript
// إرسال إشعار تليجرام للموظف
if (window.sendLeaveApprovalTelegram) {
    await sendLeaveApprovalTelegram(request, status, notes);
}
```

### 5️⃣ اختبار النظام (2 دقائق)

1. افتح `hr-management.html`
2. افتح Console (F12)
3. قدّم طلب إجازة
4. **يجب يوصل إشعار للمدير في تليجرام خلال 1-2 ثانية!** 📱
5. وافق على الطلب
6. **يجب يوصل إشعار للموظف في تليجرام!** 📱

## 🎯 المشاكل وحلولها

### ❌ Bot Token not configured
- **السبب:** لم تغير Bot Token في telegram-notifications.js
- **الحل:** افتح الملف وضع Token من @BotFather

### ❌ No employee Telegram chat ID found
- **السبب:** الموظف ما عنده Chat ID في قاعدة البيانات
- **الحل:** الموظف يفتح @userinfobot ويعطيك Chat ID وتضيفه بـ SQL

### ❌ Chat not found
- **السبب:** Chat ID خطأ أو الموظف ما فتح البوت
- **الحل:** الموظف يفتح البوت الخاص بك ويضغط /start

### ✅ كيف أتأكد البوت شغال؟
افتح هذا الرابط في المتصفح (غيّر TOKEN):
```
https://api.telegram.org/bot123456789:ABCdefGHI/getMe
```
إذا طلع JSON بمعلومات البوت، يعني شغال! ✅

## 📋 إضافة واجهة إدارة Chat IDs في admin.html

أضف هذا في صفحة **إدارة الإيميلات والمديرين**:

```html
<!-- إضافة عمود Chat ID للجدول -->
<th>Chat ID</th>

<!-- إضافة الخلية في كل صف -->
<td>
    <span id="telegram-${employee.employee_id}">${employee.telegram_chat_id || 'غير محدد'}</span>
    <button onclick="editEmployeeTelegram('${employee.employee_id}', '${employee.name}', '${employee.telegram_chat_id || ''}')">
        تعديل Chat ID
    </button>
</td>
```

أضف الدالة:

```javascript
async function editEmployeeTelegram(employeeId, employeeName, currentChatId) {
    const newChatId = prompt(
        `تعديل Telegram Chat ID للموظف: ${employeeName}\n\n` +
        `Chat ID الحالي: ${currentChatId || 'غير محدد'}\n\n` +
        `أدخل Chat ID الجديد (أرقام فقط):\n` +
        `للحصول عليه: افتح @userinfobot في تليجرام`,
        currentChatId || ''
    );
    
    if (newChatId === null) return;
    
    const chatId = newChatId.trim();
    
    // التحقق من الصيغة (يجب يكون أرقام فقط)
    if (chatId && !/^\d+$/.test(chatId)) {
        alert('❌ Chat ID يجب يكون أرقام فقط!\n\nمثال: 987654321');
        return;
    }
    
    try {
        const { error } = await supabase
            .from('employees')
            .update({ telegram_chat_id: chatId || null })
            .eq('employee_id', employeeId);
        
        if (error) throw error;
        
        alert('✅ تم تحديث Chat ID بنجاح!');
        
        // تحديث الواجهة
        const element = document.getElementById(`telegram-${employeeId}`);
        if (element) {
            element.textContent = chatId || 'غير محدد';
        }
        
    } catch (error) {
        console.error('Error updating Chat ID:', error);
        alert('❌ فشل تحديث Chat ID: ' + error.message);
    }
}
```
