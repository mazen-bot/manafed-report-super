# 🔐 دليل نظام الصلاحيات الشامل

## نظرة عامة
نظام صلاحيات متطور وشامل يتحكم في وصول الموظفين إلى ميزات ووظائف مختلفة في النظام.

---

## 📋 قائمة الصلاحيات

### 1. **👁️ عرض التقارير** (viewReports)
- **الوصف**: السماح للموظف بعرض قائمة التقارير والبيانات
- **التأثير**: الوصول الأساسي لقسم التقارير
- **الافتراضي**: مفعل لجميع الموظفين

### 2. **➕ إضافة مبيعات** (addSales)
- **الوصف**: السماح بإضافة عمليات بيع جديدة
- **التأثير**: تفعيل زر "إضافة مبيعات" وإمكانية إدخال بيانات جديدة
- **الافتراضي**: مفعل لجميع الموظفين

### 3. **✏️ تعديل مبيعات** (editSales)
- **الوصف**: السماح بتعديل عمليات البيع الموجودة
- **التأثير**: تفعيل أزرار التعديل والسماح بحفظ التغييرات
- **الافتراضي**: معطل

### 4. **🗑️ حذف مبيعات** (deleteSales)
- **الوصف**: السماح بحذف عمليات البيع من النظام
- **التأثير**: تفعيل أزرار الحذف (يتطلب أيضاً صلاحية manageEmployees)
- **الافتراضي**: معطل

### 5. **👥 إدارة موظفين** (manageEmployees)
- **الوصف**: السماح بإدارة بيانات الموظفين والصلاحيات
- **التأثير**: الوصول الكامل إلى لوحة الإدارة (admin.html)
- **الافتراضي**: معطل
- **ملاحظة**: متطلب أساسي لحذف السجلات والبيانات الحساسة

### 6. **🏢 إدارة فروع** (manageBranches)
- **الوصف**: السماح بإضافة وتعديل الفروع
- **التأثير**: تفعيل قسم "إضافة فرع" في لوحة الإدارة
- **الافتراضي**: معطل

### 7. **📤 تصدير بيانات** (export)
- **الوصف**: السماح بتصدير البيانات إلى ملفات Excel و CSV
- **التأثير**: تفعيل زر "تصدير Excel" وجميع أزرار التصدير
- **الافتراضي**: معطل

### 8. **💾 نسخ احتياطية** (backup)
- **الوصف**: السماح بإنشاء واستعادة النسخ الاحتياطية
- **التأثير**: تفعيل أزرار "تحميل نسخة احتياطية" و "رفع نسخة احتياطية"
- **الافتراضي**: معطل

### 9. **📊 عرض تحليلات** (viewAnalytics)
- **الوصف**: السماح بعرض الإحصائيات والتحليلات المتقدمة
- **التأثير**: الوصول إلى قسم التحليلات والرسوم البيانية
- **الافتراضي**: معطل

### 10. **💰 إدارة كاش** (manageCash)
- **الوصف**: السماح بإدارة تسليم الكاش والتسويات
- **التأثير**: تفعيل قسم "إدارة الكاش" والتسليمات
- **الافتراضي**: معطل

### 11. **🌐 عرض جميع الفروع** (viewAllBranches)
- **الوصف**: السماح بعرض بيانات جميع الفروع
- **التأثير**: إظهار قائمة تصفية الفروع (الموظف العادي يرى فرعه فقط)
- **الافتراضي**: معطل

### 12. **🔒 بيانات حساسة** (viewSensitiveData)
- **الوصف**: السماح بعرض البيانات الحساسة (مصاريف، سحوبات، إلخ)
- **التأثير**: عرض الأعمدة والتقارير الحساسة
- **الافتراضي**: معطل

### 13. **⚙️ إعدادات النظام** (manageSettings)
- **الوصف**: السماح بتغيير إعدادات النظام العامة
- **التأثير**: الوصول إلى قسم الإعدادات والتكوين
- **الافتراضي**: معطل

## 🔧 كيفية إدارة الصلاحيات

### الطريقة 1: عند إنشاء موظف جديد (في admin.html)

1. اذهب إلى صفحة **إدارة الفروع والموظفين**
2. قم بتسجيل الدخول برقم موظف وكلمة مرور قارة (مسؤول)
3. املأ بيانات الموظف الجديد:
   - اسم الموظف
   - الرقم الوظيفي
   - اختر الفرع
   - اكتب كلمة المرور

4. اختر الصلاحيات من القائمة:
   - الصلاحيات الافتراضية: `viewReports` و `addSales` (مفعلة دائماً)
   - أضف صلاحيات إضافية حسب الحاجة

5. اضغط **حفظ الموظف**

### الطريقة 2: إدارة الصلاحيات للموظفين الموجودين

1. اذهب إلى قسم **⚙️ إدارة صلاحيات الموظفين** (في admin.html)
2. ستجد جدول يعرض جميع الموظفين وصلاحياتهم الحالية
3. لكل موظف يمكنك:
   - **تعديل**: اضغط "تعديل" لفتح نافذة تعديل الصلاحيات
   - **إعادة تعيين**: اضغط "إعادة تعيين" لإعادة الصلاحيات الافتراضية

### الطريقة 3: تحديث سريع للصلاحيات

1. في نافذة تعديل الموظف عند إنشاء/تحديث:
   - اختر من قائمة الـ 13 صلاحية
   - عرض منطقة مرآة توضح الصلاحيات المختارة
   - اضغط حفظ للتطبيق

## 🛡️ نقاط الأمان والتحكم

### نقاط التحقق:
1. **التثقق من حذف السجلات**:
   - يتطلب: `deleteSales` + `manageEmployees`
   - تأكيد المستخدم مطلوب
   - تسجيل في السجلات

2. **التثقق من التصدير**:
   - يتطلب: `export`
   - تنبيه تطبيقي عند النجاح

3. **التثقق من النسخ الاحتياطية**:
   - يتطلب: `backup`
   - تسجيل اسم المستخدم وبيانات الاستعادة

4. **التثقق من البيانات الحساسة**:
   - يتطلب: `viewSensitiveData`
   - إخفاء الأعمدة الحساسة للمستخدمين بدون الصلاحية

## 🔍 كيفية تفعيل الصلاحيات برمجياً

### في ملف reports.html:

```javascript
// التحقق من صلاحية واحدة
if (hasPermission('editSales')) {
    // السماح بالتعديل
}

// التحقق من صلاحيات متعددة (AND)
if (hasAllPermissions('editSales', 'export')) {
    // السماح بالعمليات المتقدمة
}

// التحقق من أي صلاحية (OR)
if (hasAnyPermission('manageEmployees', 'manageBranches')) {
    // السماح بالوصول الإداري
}

// عرض رسالة الرفض
if (!hasPermission('backup')) {
    showNoPermissionMessage('إنشاء نسخ احتياطية');
}

// تحديث الواجهة تلقائياً
updateUIByPermissions(); // بعد تسجيل الدخول
```

## 🚀 أفضل الممارسات

1. **أصل الحقل الأقل**: أعط الموظفين أقل صلاحيات مطلوبة فقط
2. **مراجعة دورية**: تحقق من الصلاحيات والوصول بشكل دوري
3. **تسجيل العمليات**: احتفظ بسجل للعمليات الحساسة

5. **التدريب**: دريب الموظفين على استخدام الصلاحيات برسؤولية

## 🐛 استكشاف الأخطاء

### المشكلة: لا أرى خيار الصلاحية
**الحل**: تحقق من أن لديك صلاحية `manageEmployees`

### المشكلة: الصلاحيات لا تنطبق فوراً
**الحل**: قم بتسجيل الدخول مرة أخرى لتحديث الصلاحيات

### المشكلة: رسالة "صلاحية غير كافية"
**الحل**: تواصل مع المسؤول لطلب الصلاحيات المطلوبة
