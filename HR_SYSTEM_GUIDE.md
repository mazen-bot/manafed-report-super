## 🎯 نظرة عامة

نظام إدارة الموارد البشرية هو حل متكامل لإدارة:
- ✅ طلبات الإجازات (عادية، مرضية، استئذان)
- ⚠️ إنذارات الموظفين
- 📊 تتبع رصيد الإجازات
- 👥 إدارة الطلبات والموافقات

---

## ✨ المميزات

### للموظفين 👨‍💼
- **تقديم الطلبات:**
  - إجازة عادية (سنوية)
  - إجازة مرضية
  - استئذان من الدوام
  
- **متابعة الطلبات:**
  - عرض حالة كل طلب (معلق/مقبول/مرفوض)
  - تاريخ كامل للطلبات السابقة
  
- **رصيد الإجازات:**
  - عرض الرصيد المتبقي من الإجازات السنوية
  - حساب تلقائي للأيام المستخدمة
  
- **سجل الإنذارات:**
  - الاطلاع على جميع الإنذارات الصادرة
  - تفاصيل كل إنذار (المستوى، السبب، التاريخ)

### للمدراء 👔
- **إدارة الطلبات:**
  - عرض جميع طلبات الموظفين
  - الموافقة أو رفض الطلبات
  - تصفية حسب الفرع أو الحالة
  
- **إدارة الإنذارات:**
  - إضافة إنذارات للموظفين
  - اختيار مستوى الإنذار (تنبيه، إنذار أول، ثاني، نهائي)
  - توثيق الأسباب بالتفصيل
  
- **الإحصائيات:**
  - عدد الطلبات المعلقة
  - إحصائيات شاملة لكل موظف

## 🚀 التثبيت والإعداد

### الخطوة 1: إعداد قاعدة البيانات

1. **افتح Supabase Dashboard**
   - اذهب إلى: https://supabase.com/dashboard
   - اختر مشروعك

2. **تنفيذ SQL Script**
   ```sql
   -- انسخ محتوى ملف create_hr_tables.sql
   -- الصقه في SQL Editor
   -- اضغط Run
   ```

3. **التحقق من الجداول**
   - تأكد من إنشاء الجداول:
     - `leave_requests` (طلبات الإجازات)
     - `employee_warnings` (الإنذارات)

### الخطوة 2: تفعيل الصلاحيات

تأكد من أن جدول `employees` يحتوي على الصلاحية التالية:
- `manageEmployees` - للمدراء (للموافقة على الطلبات وإضافة الإنذارات)

مثال:
```sql
UPDATE employees 
SET permissions = ARRAY['viewReports', 'manageEmployees']
WHERE employee_id = 'M001'; -- رقم المدير
```

### الخطوة 3: فتح الصفحة

افتح ملف `hr-management.html` في المتصفح:
```
file:///path/to/hr-management.html
```

أو ارفعه على استضافة ويب.

## 👨‍💼 دليل الموظف

### تسجيل الدخول
1. أدخل **الرقم الوظيفي**
2. أدخل **كلمة المرور**
3. اضغط **تسجيل الدخول**

### تقديم طلب إجازة جديد

1. **اختر تبويب "طلب جديد"**
2. **املأ البيانات:**
   - نوع الطلب (إجازة عادية/مرضية/استئذان)
   - من تاريخ
   - إلى تاريخ
   - السبب/الملاحظات
3. **اضغط "إرسال الطلب"**

✅ سيتم إرسال الطلب للإدارة تلقائياً

### متابعة طلباتي

في تبويب **"طلباتي"**:
- 🟡 **معلق**: قيد المراجعة
- 🟢 **مقبول**: تمت الموافقة
- 🔴 **مرفوض**: تم الرفض

### عرض إنذاراتي

في تبويب **"إنذاراتي"**:
- عرض جميع الإنذارات الصادرة
- المستوى + السبب + التاريخ

## 👔 دليل المدير

### إدارة جميع الطلبات

1. **اختر تبويب "جميع الطلبات"**
2. **عرض جميع طلبات الموظفين مع تفاصيلها:**
   - اسم الموظف
   - الفرع
   - نوع الطلب
   - التواريخ
   - السبب
   - الحالة
3. **اتخاذ قرار:**
   - ✅ **قبول** - الموافقة على الطلب
   - ❌ **رفض** - رفض الطلب

### إضافة إنذار لموظف

1. **اختر تبويب "إضافة إنذار"**
2. **املأ البيانات:**
   - اختر الموظف
   - مستوى الإنذار:
     - تنبيه (خفيف)
     - إنذار أول
     - إنذار ثاني
     - إنذار نهائي (قبل الفصل)
   - سبب الإنذار (بالتفصيل)
3. **اضغط "إضافة الإنذار"**

⚠️ **ملاحظة:** سيظهر الإنذار مباشرة في سجل الموظف

## 🗄️ هيكل قاعدة البيانات

### جدول `leave_requests`

| الحقل | النوع | الوصف |
|-------|------|-------|
| id | BIGSERIAL | المعرّف الفريد |
| employee_id | TEXT | الرقم الوظيفي |
| employee_name | TEXT | اسم الموظف |
| branch | TEXT | الفرع |
| request_type | TEXT | نوع الطلب (vacation/sick/permission) |
| from_date | DATE | من تاريخ |
| to_date | DATE | إلى تاريخ |
| days_count | INTEGER | عدد الأيام |
| reason | TEXT | السبب |
| status | TEXT | الحالة (pending/approved/rejected) |
| approved_by | TEXT | رقم المعتمد (اختياري) |
| approved_at | TIMESTAMP | تاريخ الاعتماد |
| created_at | TIMESTAMP | تاريخ الإنشاء |

### جدول `employee_warnings`

| الحقل | النوع | الوصف |
|-------|------|-------|
| id | BIGSERIAL | المعرّف الفريد |
| employee_id | TEXT | الرقم الوظيفي |
| warning_level | TEXT | مستوى الإنذار |
| reason | TEXT | السبب |
| issued_by | TEXT | رقم المصدر |
| issued_by_name | TEXT | اسم المصدر |
| created_at | TIMESTAMP | تاريخ الإصدار |

## ❓ الأسئلة الشائعة

### كم رصيد الإجازات السنوية؟
- **30 يوم** افتراضياً (يمكن تعديله في الإعدادات)

### هل يمكن تعديل طلب بعد إرساله؟
- ❌ لا، يجب تقديم طلب جديد أو التواصل مع الإدارة

### متى تُعتمد الطلبات؟
- يعتمد على سياسة الشركة، عادة خلال 1-3 أيام عمل

### هل الإنذارات دائمة؟
- الإنذارات تُسجل في النظام لكن يمكن للإدارة حذفها

### كيف أعرف إذا تمت الموافقة على طلبي؟
- تحقق من تبويب "طلباتي" - ستجد الحالة محدثة
