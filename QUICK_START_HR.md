# 🚀 دليل البدء السريع - نظام الموارد البشرية
## Quick Start Guide

---

## ⚡ 3 خطوات للبدء

### 1️⃣ إنشاء الجداول في Supabase

1. افتح **Supabase Dashboard**: https://supabase.com/dashboard
2. اذهب إلى **SQL Editor**
3. انسخ محتوى ملف `create_hr_tables.sql`
4. الصقه في المحرر
5. اضغط **Run** / **تشغيل**

✅ سيتم إنشاء:
- جدول `leave_requests` (طلبات الإجازات)
- جدول `employee_warnings` (الإنذارات)

---

### 2️⃣ تفعيل صلاحيات المدراء

قم بتشغيل هذا الأمر لإعطاء صلاحية المدير:

```sql
UPDATE employees 
SET permissions = ARRAY['viewReports', 'manageEmployees', 'viewAllBranches']
WHERE employee_id = 'M001'; -- غيّر M001 برقم المدير
```

---

### 3️⃣ فتح النظام

افتح ملف `hr-management.html` في المتصفح:
```
file:///path/to/hr-management.html
```

أو ارفعه على استضافة ويب.

---

✅ **index.html** - صفحة المبيعات

## 👥 الاستخدام السريع

### للموظف:
1. سجل الدخول بالرقم الوظيفي وكلمة المرور
2. اذهب إلى **"طلب جديد"**
3. اختر نوع الطلب وأدخل التفاصيل
4. اضغط **"إرسال الطلب"**

### للمدير:
1. سجل الدخول
2. اذهب إلى **"جميع الطلبات"**
3. اضغط **✓ قبول** أو **✗ رفض**
4. يمكنك أيضاً إضافة إنذارات من تبويب **"إضافة إنذار"**

## 🔧 المشاكل الشائعة

### المشكلة: "لا يمكن الاتصال بقاعدة البيانات"
**الحل:** تأكد من:
- تشغيل أوامر SQL في Supabase
- صحة بيانات الاتصال في الكود (supabaseUrl & supabaseAnonKey)

### المشكلة: "لا أرى تبويب 'جميع الطلبات'"
**الحل:** تحتاج صلاحية `manageEmployees` في جدول employees

### المشكلة: "خطأ عند إرسال الطلب"
**الحل:** تأكد من:
- إنشاء الجداول بنجاح
- تفعيل Row Level Security (تم إضافته في SQL Script)

3. **من صفحة الإدارة (admin.html):**
