## الاستخدام السريع

### التثبيت
أضف هاتين السطرين في `<head>` لأي صفحة:
```html
<script src="custom-alerts.js"></script>
<script src="override-alerts.js"></script>
```

### الأمثلة

#### تنبيه بسيط
```javascript
showCustomSuccess('تم الحفظ بنجاح', 'تم');
```

#### تأكيد الحذف
```javascript
const confirmed = await confirm('هل أنت متأكد من الحذف؟');
if (confirmed) {
    // عملية الحذف
}
```

#### إشعار خطأ
```javascript
showCustomError('حدث خطأ في الاتصال', 'خطأ الاتصال');
```

## ملاحظات تقنية

⚠️ **مهم**: جميع استخدامات `confirm()` و `prompt()` تعود الآن Promises، لذا يجب استخدام `await`:

```javascript
// ✅ الطريقة الصحيحة
const result = await confirm('هل تريد المتابعة؟');

// ❌ لن تعمل كما هو متوقع
const result = confirm('هل تريد المتابعة؟');
```
